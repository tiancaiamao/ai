## 最高价值发现

1. **文件路径问题是最大系统性问题** — 6/23 失败任务(26%)都是因为 Agent 将文件写到 setup/ 而非 WORK_DIR 根或 /app/。这是最容易修复的,修复后通过率可从 42.5% 提升到 57.5%。

2. **regex-chess 的输出 token 耗尽** — 模型在单次推理中产生了 65K tokens 全部用于思考,没有任何工具调用输出。需要限制思考长度或增加输出预算。

3. **count-dataset-tokens 测试本身有 bug** — 测试代码写法错误 (运算符优先级),不是 Agent 的问题。

4. **Agent_* 任务全部因约束失败** — 但功能测试全部通过,说明 Agent 能力足够但约束检查过于严格或 Agent 行为不符合约束规范。
